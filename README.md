| Control | Function |
|------|------|

# Addon System

Addon packs are simple JSON-driven packages.

Example addon structure:

```
AetherAddon_Agartha/
 ├ addon.json
 ├ panel.json
 ├ layout.json
 ├ presets.json
 ├ assets/
 │  ├ background.png
 │  ├ orb.png
 │  └ glyphs.png
```

The engine validates everything automatically.

If an addon breaks, AETHER safely falls back to the default theme.

3. Replace art assets
4. Run validator
5. Package addon

Release it however you want.

# Installation

Download the latest release and place plugin files in your plugin directory.

Mac:

```
/Library/Audio/Plug-Ins/VST3
/Library/Audio/Plug-Ins/Components
```

Windows:

```
C:\Program Files\Common Files\VST3
```

Restart your DAW.
